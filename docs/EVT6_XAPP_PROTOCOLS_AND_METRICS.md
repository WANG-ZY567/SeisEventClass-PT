# EVT6 Extended Protocols and Metrics

This document summarizes auxiliary evaluation protocols for EVT6 experiments.

## Evaluation Protocols

- sample-level classification;
- event-level probability aggregation;
- few-shot evaluation;
- cross-catalog or zero-shot transfer;
- class-wise sensitivity analysis;
- picking-conditioned or routed evaluation.

## Metrics

Recommended metrics include accuracy, macro F1, weighted F1, per-class precision, per-class recall, confusion matrices, and event-level aggregated performance.

## Reporting Practice

Report the data split, number of samples, number of events, model checkpoint, random seed, and aggregation rule for each experiment.
